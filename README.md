# Fubaza - Sports Management Platform

## Overview

Fubaza is a lightweight sports management backend built with .NET 8 and ASP.NET Core. It provides REST APIs and backend services to manage clubs, players, events, templates, notifications, and background jobs. The project follows Clean Architecture principles to keep business logic, infrastructure, and presentation concerns separated for easier maintenance, testing, and scaling.

## Technology Stack (brief)

- Framework: .NET 8 / ASP.NET Core 8  
- Language: C# 12  
- ORM: Entity Framework Core 8  
- Databases: SQL Server or PostgreSQL  
- Background jobs: Hangfire  
- Logging: Serilog  
- API docs: Swagger / OpenAPI  
- Mapping & Validation: AutoMapper, FluentValidation
